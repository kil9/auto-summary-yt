# SPEC

## 목표
- 크롬 확장 이름: auto-summary-yt
- 유튜브 시청 중 작은 버튼 또는 단축키로 동작
- 크롬의 분할 기능을 이용해 Gemini 웹사이트를 띄움
- 현재 보고 있는 유튜브 URL을 Gemini 입력창에 자동 입력
- API 사용 없음

## 주요 기능
- 콘텐츠 스크립트로 유튜브 페이지에 작은 버튼 표시
- 단축키로도 동일 기능 실행
- 크롬 분할(스플릿) 기능을 이용해 Gemini 페이지를 함께 표시
- 현재 탭의 유튜브 URL 자동 복사/주입

## 설정 페이지
- 단축키/버튼 표시 등 기본 옵션 설정 UI 제공

## 기술 스택
- 크롬 확장: Manifest V3
- UI: 표준 Web APIs
- 빌드 도구: 필요한 경우 최신 안정 버전으로 선택
