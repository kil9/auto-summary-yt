# auto-summary-yt

유튜브 시청 중 버튼 또는 단축키로 Gemini 웹을 분할 화면으로 열고, 현재 유튜브 URL을 자동 입력하는 크롬 확장입니다. API를 사용하지 않습니다.

## 주요 기능

- 유튜브 영상 제목 옆에 "Gemini" 버튼 표시
- 단축키 `Ctrl+Shift+U` (Mac: `Command+Shift+U`)로 동일 기능 실행
- 현재 창을 좌우로 나누거나 백그라운드 탭으로 Gemini 열기
- 현재 유튜브 URL을 Gemini 입력창에 자동 입력 후 전송
- 옵션 페이지에서 버튼 표시 / URL 자동 입력 / 탭 열기 방식 설정 가능

## 설치 방법

1. GitHub 우측의 **Releases**에서 최신 릴리즈의 zip 파일을 내려받아 압축을 풉니다.
2. 크롬에서 `chrome://extensions`로 이동합니다.
3. 우측 상단의 **개발자 모드**를 켭니다.
4. **압축해제된 확장 프로그램을 로드합니다**를 선택하고 압축을 푼 폴더를 지정합니다.

## 개발 중 로드 방법

별도 빌드 도구 없이 소스 디렉터리를 직접 로드할 수 있습니다.

1. 저장소를 클론합니다.
2. `chrome://extensions` → 개발자 모드 → **압축해제된 확장 프로그램을 로드합니다**
3. 저장소 루트 폴더를 선택합니다.
4. 코드 수정 후 확장 프로그램 페이지에서 새로고침 아이콘을 클릭하면 반영됩니다.
