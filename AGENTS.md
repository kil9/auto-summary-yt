# AGENTS

## 프로젝트 목적
- 유튜브 시청 중 버튼/단축키로 Gemini 웹을 분할 화면으로 열고 URL을 자동 입력하는 크롬 확장.

## 핵심 제약
- API 사용 금지(웹 UI 자동화 기반).
- 크롬 확장 Manifest V3 기준.
- 최신 안정 버전의 일반적인 웹 기술 사용.

## 동작 흐름(개요)
1. 유튜브 페이지에서 작은 버튼을 제목 오른쪽에 표시.
2. 버튼 클릭 또는 단축키(`Ctrl+Shift+U`) 입력.
3. 현재 창을 좌우로 나누고 Gemini 팝업 창을 띄움.
4. 현재 유튜브 URL을 Gemini 입력창에 자동 입력.

## 구현 포인트
- 콘텐츠 스크립트: 유튜브 페이지 감지, 버튼 렌더링, URL 수집.
- 백그라운드/서비스 워커: 단축키 처리, 탭/창 제어.
- 옵션 페이지: 버튼 표시, URL 자동 입력 옵션.

## 파일 구조
- `manifest.json`
- `src/background/`
- `src/content/`
- `src/options/`
